개인정보처리방침

한화 스코어 알림 (Hanwha Score Alert) Chrome Extension
✓ 개인정보 수집 없음

한화 스코어 알림 확장 프로그램은 사용자의 개인정보를 일절 수집하지 않습니다. 이 확장 프로그램은 KBO 한화 이글스의 실시간 경기 점수를 표시하는 것이 유일한 목적입니다.

수집하는 정보

이 확장 프로그램은 사용자 개인을 식별할 수 있는 어떠한 정보도 수집하지 않습니다.

  • 이름, 이메일, 전화번호 등 개인 식별 정보 — 수집 안 함
  • 위치 정보 — 수집 안 함
  • 브라우저 방문 기록 — 수집 안 함
  • 쿠키 또는 추적 데이터 — 사용 안 함
  • 분석(Analytics) 데이터 — 수집 안 함

로컬 저장 데이터

확장 프로그램은 다음 정보를 사용자의 Chrome 브라우저에만 저장합니다. 외부 서버로 전송되지 않습니다.

  • chrome.storage.sync — 알림 설정(켜기/끄기) 등 사용자 환경설정. Chrome 계정을 통해 사용자 본인의 기기 간에만 동기화됩니다.
  • chrome.storage.local — 마지막으로 확인한 점수 상태. 중복 알림 방지를 위해 기기 내에만 저장됩니다.

외부 API 호출

확장 프로그램은 KBO 경기 점수를 가져오기 위해 네이버 스포츠 공개 API를 호출합니다.

  • 호출 주소: api-gw.sports.naver.com
  • 요청 내용: 오늘의 KBO 경기 일정 및 점수 (공개 데이터)
  • 사용자 개인정보는 해당 요청에 포함되지 않습니다.
  • 해당 API 이외의 서버로는 어떠한 데이터도 전송하지 않습니다.

제3자 데이터 공유

수집된 개인정보가 없으므로, 제3자에게 공유되는 정보도 없습니다.

권한 사용 목적

  • alarms — 주기적으로 경기 점수를 폴링하기 위해 사용
  • notifications — 득점 및 경기 시작/종료 시 데스크톱 알림 표시
  • storage — 알림 설정 및 마지막 점수 상태를 로컬에 저장
  • host_permissions (api-gw.sports.naver.com) — 네이버 스포츠 API에서 경기 점수 데이터 수신
  • host_permissions (sports-phinf.pstatic.net) — 팀 로고 이미지 표시

방침 변경

본 개인정보처리방침이 변경될 경우, GitHub 저장소를 통해 공지합니다. 확장 프로그램의 기본 원칙(개인정보 미수집)은 변경되지 않습니다.
